DirtTW's Basement Optimize Pack

DBOP是一個基於 fabric模組平台的一個優化包 大部分模組都是客戶端的 並且能夠優化Minecraft麥塊

原味

版本: Fabric 1.19.4-1.20.2

All of Fabric 6 - AOF6

All of Fabric 6 是 AOF 系列的第六作，保持了前作的許多特點, 有許多任務 以及新戰鬥系統 跟各種建築可以探索。

All of Fabric 7 - AOF7

All of Fabric 7 是 AOF 系列的第七作，足跡與故事 探險開始!!!!

TNVBP

TNVBP 是一個基於 Fabric API 並使用優化模組來改善 Minecraft 效能的優化模組包 不影響遊戲體驗不改變玩法純優化 就像只裝上 Optifine 一樣只是 FPS 更高更流暢,相容單人及各大 1.18.2 以上原味生存伺服器

版本: Fabric 1.18.2-1.20.2
